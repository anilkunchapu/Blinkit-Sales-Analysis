# Blinkit Retail Sales Analysis - Power BI

## Project Overview

This project analyzes retail sales data using Power BI.
It provides insights into outlet performance, product categories, and sales trends.

## Key KPIs

* Total Sales
* Average Sales per Outlet
* Product Ratings
* Item Category Performance

## Key Insights

* Medium-sized outlets contribute the highest sales
* Tier 3 locations outperform Tier 1 and Tier 2
* Low-fat items generate the majority of sales

## Tools Used

* Power BI
* DAX
* Excel

## Business Recommendations

* Expand medium-sized outlets in Tier 3 locations
* Increase low-fat product assortment
* Focus on top-performing product categories
